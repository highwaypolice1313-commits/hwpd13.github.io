คัดลอกส่วนหัวและสคริปต์นี้ไปวางใน Google Apps Script ของท่าน เพื่อเชื่อมต่อระบบแอปพลิเคชันเข้ากับคลาวด์จริงทันที

📜 google_apps_script_backend.js
ขั้นตอนการติดตั้งและเชื่อมต่อระบบจริงโดยละเอียด:
ขั้นตอนที่ 1: การจัดการพื้นที่ฝั่ง Google Cloud (Drive & Sheets)
  1. สร้างโฟลเดอร์หลักขึ้นมาใหม่บน Google Drive ของท่าน จากนั้นให้คลิกเปิดโฟลเดอร์นั้น แล้วคัดลอก **ID โฟลเดอร์** จากแถบที่อยู่บนสุดของเบราว์เซอร์
  2. สร้าง Google Sheets ใหม่ ทำการป้อนชื่อแถวแรก (แถวที่ 1 คอลัมน์ A - I) เรียงลำดับจากซ้ายไปขวาดังนี้: `Timestamp`, `วันที่บันทึก`, `LINE User ID`, `ชื่อผู้ส่ง (LINE Name)`, `หมวดอักษร`, `หมวดตัวเลข`, `หมวดจังหวัด`, `ชื่อไฟล์`, `ลิงก์ไฟล์บน Drive`
ขั้นตอนที่ 2: การเปิดใช้และเขียนคำสั่งบน Google Apps Script
  1. บน Google Sheets ที่พึ่งสร้าง กดแถบเมนูด้านบนเลือก **"ส่วนขยาย" (Extensions)** -> **"Google Apps Script"**
  2. ลบโค้ดเดิมทิ้งให้หมด แล้วทำการคัดลอกสคริปต์จากกล่องสีดำด้านบนมาวางลงไปทั้งหมด
  3. ทำการแทนที่ข้อความ `"ใส่_ID_โฟลเดอร์หลักบนไดร์ฟของท่านที่นี่"` ในบรรทัดที่ 6 ของโค้ด ด้วย **ID โฟลเดอร์หลัก** ที่คัดลอกมาจาก Google Drive ในขั้นตอนแรก
  4. กดปุ่มบันทึก จากนั้นมองที่มุมขวาบนคลิกปุ่มสีน้ำเงิน **"ทำให้ใช้งานได้" (Deploy)** -> **"การทำให้ใช้งานได้ใหม่" (New Deployment)**
  5. เลือกประเภทการติดตั้งเป็น **"เว็บแอป" (Web App)** ตั้งค่ากำหนด 'ผู้มีสิทธิ์เข้าถึง' เป็น **"ทุกคน" (Anyone)** แล้วกด "ทำให้ใช้งานได้"
  6. ระบบจะขอให้ยืนยันสิทธิ์ในการเข้าถึง Google Drive และ Google Sheets ของคุณ ให้กดยอมรับให้สิทธิ์เรียบร้อย จากนั้นให้ **คัดลอก URL ของเว็บแอป** ที่ได้รับไว้
ขั้นตอนที่ 3: การสร้างและติดตั้ง LINE Mini App (LIFF) และผูกความปลอดภัย
  1. เข้าสู่หน้า [LINE Developers Console](https://developers.line.biz/) ล็อกอินด้วยบัญชี LINE ของท่าน
  2. สร้าง **Provider** และสร้างช่องทางแอปพลิเคชันแบบ **"LINE liff (LINE Mini App)"**
  3. ในแถบตั้งค่าของ LIFF ให้เพิ่มแอป และคัดลอก **LIFF ID** ที่ได้รับ (เช่น `200xxxxxxx-xxxxxxxx`) เก็บเอาไว้
  4. นำไฟล์ `index.html` ของระบบนี้ไปเปิดโฮสต์บนพื้นที่ความปลอดภัยประเภท HTTPS (เช่น GitHub Pages หรือ Vercel) นำ URL นั้นมากรอกลงในช่อง **Endpoint URL** ของ LINE LIFF
ขั้นตอนที่ 4: เชื่อมโยงระบบทั้งหมดและสลับเข้าสู่ Live API Mode
  1. เปิดหน้าเว็บพอร์ทัลหลักของคุณ กดปุ่มสีเทาที่มุมขวาบน **"ตั้งค่าเชื่อมต่อระบบจริง"**
  2. ป้อนข้อมูล **Apps Script Web App URL** จากขั้นตอนที่ 2 และ **LINE LIFF ID** จากขั้นตอนที่ 3 ลงไปและกดตกลง บันทึก
  3. ระบบจะสลับโหมดจาก "จำลองสถานการณ์" เข้าสู่การส่งข้อมูลคลาวด์จริงและเรียกใช้งานโปรไฟล์ LINE ของคุณจริงทันทีอย่างสมบูรณ์แบบ!

ตั้งค่าเชื่อมต่อ LINE LIFF & API คลาวด์จริง

เชื่อมต่อพอร์ตเพื่อยิง API ส่งข้อมูลจริงไปยัง Google Sheets และ Google Drive ของคุณโดยไม่ต้องผ่านการทดสอบจำลอง

⚠️ หากคุณกรอกข้อมูลครบถ้วน ระบบจะบันทึกข้อมูลแบบปลอดภัยในเครื่อง และพยายามเรียกใช้ `liff.init()` จริง และการส่งโพสต์จริงไปยังสคริปต์ของคุณ หากเว้นว่างไว้ ระบบจะยังทำงานอยู่ในสถานะ "ระบบจำลองความสำเร็จ" เพื่อให้คุณพรีวิวทดสอบระบบก่อน
